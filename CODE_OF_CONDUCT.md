# Code of Conduct

## Our standards

We are committed to a welcoming, harassment-free experience for everyone.

Examples of expected behavior include:

- Respectful, constructive communication
- Graceful acceptance of feedback
- Prioritizing community well-being

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

Report concerns to the maintainers via private channels defined in the repository settings.
